## 🛠 Подробнее о задачах


### 1. Digest
Погружение во внутреннее устройство Go и работу с памятью.
*   **Задача:** Реализовать различные утилиты, включая манипуляции с `unsafe` указателями, решение квадратных уравнений и in-place сортировки.


### 2. LFU Cache
Проектирование эффективных структур данных.
*   **Задача:** Написать кэш, который удаляет наиболее редко используемые элементы при переполнении.
*   **Особенности:**
    *   Полная поддержка **Generics** (`[K comparable, V any]`).
    *   Реализация собственного **связного списка**, так как стандартный `container/list` хранит `any` (interface{}) и не типобезопасен.
    *   Сложность всех операций (Get/Put) — **O(1)**.

### 3. Go Assembler
Низкоуровневая оптимизация критически важных участков кода.
*   **Задача:** Написать функции суммы, бинарного поиска (LowerBound) и чисел Фибоначчи напрямую на ассемблере.
